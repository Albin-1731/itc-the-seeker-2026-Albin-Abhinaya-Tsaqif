## Fitur Utama
- Halaman Beranda dengan profil ITC.
- Daftar Struktur Pengurus (Ketua, Wakil, Kadiv, Wakadiv, Aggota).
- Detail Pengurus (Tugas, Jabatan, dan Tombol Favorite).
- Struktur data menggunakan data dummy.

## Cara Menjalankan Aplikasi
Pastikan kamu sudah menginstal Flutter SDK di komputermu.

1. Clone repository ini:
   git clone https://github.com/Albin-1731/itc-the-seeker-2026-Albin-Abhinaya-Tsaqif.git

2. Masuk ke direktori proyek:
   cd itc-the-seeker-2026-albin-abhinaya-tsaqif

3. Ambil paket/dependencies yang diperlukan:
   flutter pub get

4. Jalankan aplikasi:
   flutter run

## Aplikasi didalamnya
1. masuk halaman beranda dengan profil itc
2. gulir kebawah dan ada "lihat struktur" klik
3. lanjut halaman daftar pengurus
4. disini bisa lihat detail data anggota
5. done
